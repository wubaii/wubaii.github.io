JVM学习笔记

JVM介绍

什么是JVM

  • 定义：java virtual machine–java程序的运行环境（字节码文件的运行环境）
  • JVM好处：
  1. 一次编译，到处运行（JVM实现）
  2. 自动内存管理—垃圾回收机制
  3. 数组下标越界检查
  4. 多态
  • jdk、jre、jvm比较

Hotspot

本笔记以oracle的Hotspot为例。

学习路线

JVM主要学习内容：

  1. 类加载器–将二进制字节码文件加载到JVM中运行

  2. JVM内存结构

    • 类是放在方法区中
    • 创建的对象放在堆中
    • 对象调用方法要用到虚拟机栈，本地方法栈，程序计数器

  3. 执行引擎

    • 每行代码由解释器逐行解释执行
    • JIT进行热点代码的优化执行
    • GC对堆中不在被引用的对象进行回收

  4. 本地方法接口调用操作系统功能
